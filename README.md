# icon 图标
1. svg图标

   >  注意：自定义 svg 图标固定放在 src/assets/svg 下

   使用方法：`<icon-custom-logo />`或`<svg-icon icon="logo" />`

2. iconify图标

   使用方法：`<icon-ant-design:fullscreen-exit-outlined  />`

## Options

| 属性  | 类型   | 描述             | 默认值 | 备注 |
| ----- | ------ | ---------------- | ------ | ---- |
| icon  | String | [必填]，图标名称 |        |      |
| size  | String | 图标大小         | 1em    |      |
| color | String | 图标颜色         | #333   |      |
